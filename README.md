# Igor - Wit, Web Audio API and XBMC experiment

Like everything in life, inspired by the movie [Brainscan](http://youtu.be/mT1Vr13s17U?t=25m33s) (25m 33s).

## Quickstart

### Install bower deps

```
> bower install bootstrap
> bower install angular
```

### Install node deps

```
> node install karma
```
